Задача «Pacman»
Версия 1.0
Автор: Слышева Александра
Описание
Данное приложение является реализацией игры pacman.
Состав
•	Запускаемый модуль: main_window.py
•	Модули логики игры: game_logic.py, monster.py, pacman.py, game_building.py, game_states.py
•	Модуль обработки событий и отрисовки игры: game_painter.py
•	Модуль запуска музыки music_executer.py
•	Модуль описания карты: field.py
•	Модули для оконного приложения: login.py, records_table.py
•	Директория levels для описания карт уровней и порядка их прохождения
•	Директория music для музыки
•	Дириктория images для изображений
Консольная версия
Пример запуска: python main_window.py
Управление:
←, →, ↑, ↓ - движение пакмена
P - пауза
